Projects

Project 1 - Cloud Resume Challenge

This static website represents my completion of the Cloud Resume Challenge, where I leveraged a range of AWS services to build and deploy the project. The site is developed using HTML and CSS, and enhanced with CloudFront for HTTPS security and Route 53 for managing a custom domain name. A visitor counter feature was implemented using JavaScript, with DynamoDB handling data storage and updates. To ensure secure and efficient communication with DynamoDB, I created a custom API using Python, AWS Lambda, and API Gateway. Continuous Integration and Continuous Deployment (CI/CD) were managed through a GitHub repository. Throughout the project, I encountered and overcame several challenges with unfamiliar AWS services, gaining valuable experience through persistent research and iterative problem-solving.

Project 2 - iOS App

I used Xcode to develop an iOS app for a printing company that enables customers to seamlessly upload their artwork and select specific printing requirements. By leveraging Xcode's robust development environment and Swift programming, I designed a user-friendly interface that allows customers to easily navigate through options, specify their needs, and submit their artwork directly through the app. The app streamlines the order process, ensuring that customers can efficiently communicate their printing specifications and upload files in a few simple steps.

Project 4 - Tier Game

In my project, I utilized several key skills that are critical in modern web development. I applied responsive design principles using viewport meta tags to ensure the layout adapts to different screen sizes. I employed Flexbox and CSS Grid to create a dynamic and user-friendly tier list interface. For interactivity, I implemented drag-and-drop functionality with JavaScript, allowing users to rearrange items seamlessly. My JavaScript code handles event management and data serialization, using JSON to present the final rankings effectively. The clean and modular approach to my code, along with attention to styling and user feedback, demonstrates my proficiency in front-end development and my ability to create engaging and functional web applications.
